Delete your data

How to remove local app data and optional shared collaboration data.

Local app data

Most app data, such as local shopping lists, notes, favorites, and preferences, is stored locally on your device. To delete local app data:

Android (general steps)

  1. Open Settings
  2. Go to Apps → Shopping List & Notes
  3. Open Storage
  4. Tap Clear data (and optionally Clear cache)
  5. Optionally uninstall the app to remove it completely

Shared lists and Google Sign-In data

If you use optional shared list collaboration, some data may be stored in Firebase services to provide realtime sync, invitations, roles, member information, and shared list notifications.

To stop using shared lists, you can sign out from the app settings. Signing out disables shared collaboration on that device and removes or disables the device notification token where applicable.

If you want shared collaboration data associated with your Google account to be removed, contact support using the email address below. Include the Google account email used for shared lists so the request can be identified.

Push notification data

Shared list push notifications use Firebase Cloud Messaging. You can disable notifications from Android notification settings or from the app settings where available. Signing out also disables shared-list notification delivery for that device where applicable.

Exports and backups

If you use export/import backups, deleting app data does not automatically delete files you exported elsewhere. Delete those files manually from your device storage, cloud storage, or any other location where you saved them.

Request data deletion

To request deletion of optional cloud/shared collaboration data, contact: janstech.apps@gmail.com
