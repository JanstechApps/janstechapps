# Shopping List & Notes – Legal & Support Pages

This repository hosts the official GitHub Pages site for the Android application:

**Shopping List & Notes (Kauppalista & Muistiinpanot)**

The app is designed with an offline-first and privacy-first approach.
Most features work locally on the user’s device, while optional cloud features
such as shared realtime collaboration use Firebase services.

# Main app features

- Offline-first shopping lists and notes
- Optional AI-powered features
- Optional Google Sign-In
- Optional realtime shared lists
- Shared shopping lists and task lists
- Push notifications for shared collaboration
- Export/import backups
- Privacy-first architecture
